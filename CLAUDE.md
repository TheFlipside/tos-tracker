# Project: TOS-Tracker

## What This Project Does

From a list of sources, visit various websites regarding Terms Of Service and
Privacy Policies and download them.
By tracking them via version control systems, changes over time can be tracked.

## Project-Specific Rules

Obtained files should be stored in subfolders named after the organization;
e.g. apple/ google/ microsft/ etc.

## Skills Available

- `codebase-navigator` — use when first exploring this repo
- `code-quality` — use before committing any changes
